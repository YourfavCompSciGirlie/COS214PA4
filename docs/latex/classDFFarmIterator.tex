\doxysubsection{Detailed Description}
Implements the depth-\/first traversal iterator for farm units. 

This class allows traversal of farm units using the depth-\/first search (DFS) strategy. It maintains a stack to keep track of the current path during the traversal and an unordered set to track visited farm units, ensuring that each unit is processed only once.
