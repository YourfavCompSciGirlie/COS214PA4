\doxysubsection{Detailed Description}
Concrete state class representing a flooded soil state. 

This class inherits from \mbox{\hyperlink{classSoilState}{Soil\+State}} and represents a state where the soil is flooded. It provides implementations for harvesting crops and handling rain events specific to this state.
