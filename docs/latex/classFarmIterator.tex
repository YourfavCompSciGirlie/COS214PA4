\doxysubsection{Detailed Description}
Abstract class representing the Iterator for traversing Farm\+Units. 

The \mbox{\hyperlink{classFarmIterator}{Farm\+Iterator}} class provides the interface for implementing iterators to traverse farm units using various traversal strategies, such as BFS or DFS.
