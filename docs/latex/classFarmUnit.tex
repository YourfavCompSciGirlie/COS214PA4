\doxysubsection{Detailed Description}
Abstract class representing a unit in the farm. 

The \mbox{\hyperlink{classFarmUnit}{Farm\+Unit}} class is the base component for the Composite pattern, representing individual or composite farm units. It also serves as the aggregate for the Iterator pattern, enabling traversal through farm sub-\/units.
