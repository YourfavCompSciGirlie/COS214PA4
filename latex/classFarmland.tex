\doxysubsection{Detailed Description}
Represents a composite farm unit that can contain other farm units. 

The \mbox{\hyperlink{classFarmland}{Farmland}} class implements the Composite pattern to manage a collection of \mbox{\hyperlink{classFarmUnit}{Farm\+Unit}} objects. It also implements the Iterator pattern to provide access to its children elements.
