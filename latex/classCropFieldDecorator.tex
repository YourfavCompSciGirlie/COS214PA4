\doxysubsection{Detailed Description}
Abstract base class for \mbox{\hyperlink{classCropField}{Crop\+Field}} decorators that adds additional functionality to \mbox{\hyperlink{classCropField}{Crop\+Field}} objects. 

This class implements the Decorator design pattern. It wraps a \mbox{\hyperlink{classCropField}{Crop\+Field}} object and allows for additional behavior to be added without modifying the \mbox{\hyperlink{classCropField}{Crop\+Field}} class itself.
